## 4.4. Contoh Praktis
Bayangkan departemen Keuangan perlu mengajukan Laporan Pajak Penghasilan Badan yang pertama untuk Juni 2024, untuk perusahaan Alpha Corp (A1) terkait Proyek Phoenix (PRJ007).

Kode Panggil akan disusun sebagai berikut:
- **Fungsi:** F (Keuangan)
- **Topik:** CT (Pajak Perusahaan)
- **Sub-Topik:** IR (Laporan Pajak Penghasilan)
- **Perusahaan:** A1 (Alpha Corp)
- **Fitur:** PRJ007 (Proyek Phoenix)
- **Bulan/Tahun:** J24 (Juni 2024)
- **Urutan:** 01 (Dokumen pertama jenis ini di bulan ini)

Kode Panggil lengkapnya adalah:

F-CT-IR-A1-PRJ007-J24-01


Kode ini secara instan memberi tahu siapa pun yang akrab dengan sistem tentang apa dokumen itu, milik siapa, terkait dengan apa, dan kapan dibuat.

## 4.5. Cara Menggunakan Kode Panggil di DOMAN

- **Saat Mencari:** Gunakan Kode Panggil lengkap di bilah pencarian utama untuk hasil instan, atau gunakan filter pencarian lanjutan untuk mencari berdasarkan komponen individual.
- **Saat Mengunggah/Mengarsipkan:** Sistem akan memandu Anda mengisi metadata untuk setiap komponen. Akurasi sangat penting di sini.
- **Saat Berkomunikasi:** Merujuk pada Kode Panggil dokumen menghilangkan ambiguitas dan memastikan semua orang melihat file yang benar.
